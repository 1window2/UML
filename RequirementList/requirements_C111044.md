# C111044 김호성 회원가입 회원탈퇴 로그인 로그아웃 Requirements List

| No | Requirement | Use Case(s) |
|---|---|---|
| 1 | 시스템을 이용하기 위해 사용자는 회원 가입을 해야 하며, ID, 비밀번호, 이름, 전화번호, 이메일을 필수로 입력해야 한다 | 회원가입 |
| 2 | 시스템은 회원가입 시 입력된 필수 정보의 유효성을 검증해야 하고, 유효하지 않은 경우 해당 항목을 강조표시하고 오류 메시지를 출력해야 한다 | 회원가입 |
| 3 | 시스템은 회원가입 시 필수 입력 정보가 누락된 경우 누락된 항목을 강조 표시하고 오류 메시지를 출력해야 한다 | 회원가입 |
| 4 | 시스템은 회원가입 시 입력한 ID가 이미 존재하는 경우 아이디 중복 오류 메시지를 출력해야 한다 | 회원가입 |
| 5 | 시스템은 회원가입이 완료되면 완료 메시지를 출력하고 로그인 화면으로 이동해야 한다 | 회원가입 |
| 6 | 회원은 언제든 탈퇴할 수 있다 | 회원탈퇴 |
| 7 | 시스템은 회원 탈퇴 요청 시 탈퇴 확인 경고 메시지를 출력해야 한다 | 회원 탈퇴 |
| 8 | 시스템은 회원 탈퇴가 완료되면 회원의 모든 이용 권한과 데이터를 삭제해야 한다 | 회원 탈퇴 |
| 9 | 시스템은 회원 탈퇴 완료 후 완료 메시지를 출력하고 로그인 화면으로 이동해야 한다 | 회원 탈퇴 |
| 10 | 시스템은 로그인 시 입력된 계정 정보가 등록된 정보와 일치하는지 확인해야 한다 | 로그인 |
| 11 | 시스템은 로그인에 성공한 경우 사용자 권한에 맞는 메인 화면으로 이동해야 한다 | 로그인 |
| 12 | 시스템은 로그인 정보가 일치하지 않는 경우 잘못된 필드를 강조표시하고 오류 메시지를 출력해야 한다 | 로그인 |
| 13 | 관리자는 주어진 ID(admin)와 비밀번호(admin)를 사용하여 로그인해야 한다 | 로그인 |
| 14 | 로그인한 관리자와 회원은 언제든 로그아웃할 수 있어야 한다 | 로그아웃 |
| 15 | 시스템은 로그아웃 요청 시 현재 사용자 세션을 종료하고 완료 후 로그인 화면으로 이동해야 한다 | 로그아웃 |

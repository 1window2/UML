## 회원가입

| Actor Action | System Response |
|---|---|
| 1. 회원이 회원가입 메뉴를 선택한다. | 2. 회원가입 필수 정보 입력 필드를 표시한다. |
| 3. 사용자가 ID, 비밀번호, 이름, 전화번호, 이메일 등을 입력한다. |  |
| 4. 사용자가 가입 버튼을 클릭한다. | 5. 입력된 회원 정보를 확인한다. |
|  | 6. 회원가입 완료 메시지를 출력한다. |
|  | 7. 로그인 화면으로 이동한다. |

### Alternative Course
* 4a. 필수 입력 정보가 누락된 경우 누락된 항목에 대한 오류 메시지를 출력한다.

* 4b. 이미 존재하는 ID인 경우 중복된 ID라는 메시지를 출력한다.

## 설문 등록

| Actor Action | System Response |
|---|---|
| 1. 관리자가 설문 등록 메뉴를 선택한다. | 2. 설문 정보 입력 화면을 출력한다. |
| 3. 관리자가 설문 제목, 설명, 문항, 응답 항목, 시작 시각, 마감 시각을 입력한다. |  |
| 4. 관리자가 등록 버튼을 클릭한다. | 5. 입력된 설문 정보를 확인한다. |
|  | 6. 설문 등록 완료 메시지를 출력한다. |

### Alternative Courses
* 5a. 필수 입력 정보가 누락된 경우 오류 메시지를 출력한다.
* 5b. 마감 시각이 유효하지 않은 경우 (마감시간이 과거이거나 등등) 오류 메시지를 출력한다.

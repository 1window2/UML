# 온라인 설문조사 플랫폼 Requirement List (응답 및 통계 관리)

| No | Requirement | Use Case(s) |
| :-- | :--- | :--- |
| 1 | 회원은 자신이 응답한 모든 설문 정보를 한 화면에서 조회할 수 있다 | 응답 설문 조회 |
| 2 | 시스템은 회원이 응답한 설문 정보를 조회하는 경우, 각 응답한 설문에 대해 설문 제목, 응답 내용, 응답 일시 등을 출력해야 한다 | 응답 설문 조회 |
| 3 | 시스템은 회원이 응답한 설문 내역이 없을 경우, 내역이 없다는 안내 메시지를 출력해야 한다 | 응답 설문 조회 |
| 4 | 회원은 현재 진행 중인 설문에 대해서 자신이 제출한 응답을 수정할 수 있으며, 수정 버튼을 눌러 완료한다 | 응답 수정 |
| 5 | 시스템은 설문 수정 화면에 가장 최근에 수정한 날짜를 출력해야 한다 | 응답 수정 |
| 6 | 시스템은 응답 수정 시 필수 항목이 누락되거나 지워진 경우, 데이터 저장을 거부하고 경고 메시지를 출력해야 한다 | 응답 수정 |
| 7 | 회원은 현재 진행 중인 설문에 대해서는 응답을 취소할 수 있다 | 응답 취소 |
| 8 | 시스템은 회원이 응답 취소를 요청할 경우, 데이터 삭제 경고 팝업을 출력하여 삭제 여부를 재확인해야 한다 | 응답 취소 |
| 9 | 시스템은 설문이 이미 종료된 경우에는 회원이 응답 내용을 수정하거나 취소할 수 없도록 제한해야 한다 | 응답 수정 / 응답 취소 |
| 10 | 관리자가 검색 단위를 최근 1주일, 1개월, 1년 중에서 선택하면 설문 응답 통계 정보를 조회할 수 있다 | 설문 통계 조회 |
| 11 | 시스템은 관리자가 선택한 기간 동안의 전체 설문 수, 종료된 설문 수, 총 응답 수를 출력해야 한다 | 설문 통계 조회 |

# 온라인 설문조사 플랫폼 Requirements List

## 1. 개요

본 문서는 온라인 설문조사 플랫폼의 Functional Requirement를 정의한다.

---

## 2. Functional Requirements

### 온라인 설문조사 플랫폼 Requirement List

| No | Requirement | Use Case(s) |
| :-- | :--- | :--- |
| 1 | 시스템을 이용하기 위해 사용자는 회원 가입을 해야 하며, ID, 비밀번호, 이름, 전화번호, 이메일을 필수로 입력해야 한다 | 회원가입 |
| 2 | 회원은 언제든 탈퇴할 수 있으며, 탈퇴 시 모든 이용 권한과 데이터가 삭제된다 | 회원 탈퇴 |
| 3 | 관리자와 회원은 ID와 비밀번호를 입력하여 로그인할 수 있다 (단, 관리자는 주어진 ID(admin)와 비밀번호(admin)를 사용한다) | 로그인 |
| 4 | 로그인한 관리자와 회원은 언제든 로그아웃할 수 있다 | 로그아웃 |
| 5 | 관리자는 설문 제목, 설문 설명, 설문 문항, 응답 항목, 설문 시작 시각, 설문 마감 시각 등을 입력하여 설문 정보를 등록할 수 있다 | 설문 등록 |
| 6 | 관리자는 등록된 모든 설문 리스트를 조회할 수 있다 | 설문 리스트 조회 |
| 7 | 관리자는 설문 리스트 조회 화면에서 특정 설문 항목을 선택하여 상세 내용을 팝업 창으로 볼 수 있다 | 설문 상세 조회 |
| 8 | 관리자는 설문 리스트 조회 화면에서 특정 설문 항목을 선택해서 삭제할 수 있다 | 설문 삭제 |
| 9 | 회원은 현재 진행 중인 설문, 향후 진행 예정인 설문, 종료된 설문 리스트 중 하나를 선택한 후 키워드를 입력하여 설문을 검색할 수 있다 | 설문 검색 |
| 10 | 시스템은 검색 결과로 조건에 맞는 설문 리스트를 출력해야 한다 | 설문 검색 |
| 11 | 회원은 검색된 설문 리스트에서 특정 설문을 선택하면 새로운 화면에서 상세 정보를 볼 수 있다 | 상세 정보 조회 |
| 12 | 시스템은 회원이 특정 설문을 선택하면 설문 제목, 설문 설명, 설문 문항, 응답 항목, 설문 시작 시각, 설문 마감 시각 등을 출력해야 한다 | 상세 정보 조회
| 13 | 시스템은 회원이 선택한 설문이 현재 진행 중인 경우, 응답 버튼을 활성화해야 한다 | 설문 상세정보 조회, 설문 응답 |
| 14 | 회원이 응답 버튼을 누르면 응답 화면으로 이동하여 설문에 응답할 수 있으며, 응답 버튼을 눌러 제출을 완료한다 | 설문 응답 |
| 15 | 회원은 자신이 응답한 모든 설문 정보를 한 화면에서 조회할 수 있다 | 응답 설문 조회 |
| 16 | 시스템은 회원이 응답한 설문 정보를 조회하는 경우, 각 응답한 설문에 대해 설문 제목, 응답 내용, 응답 일시 등을 출력해야 한다 | 응답 설문 조회 |
| 17 | 회원은 현재 진행 중인 설문에 대해서 자신이 제출한 응답을 수정할 수 있으며, 수정 버튼을 눌러 완료한다 | 응답 수정 |
| 18 | 시스템은 설문 수정 화면에 가장 최근에 수정한 날짜가 출력해야 한다 | 응답 수정 |
| 19 | 회원은 현재 진행 중인 설문에 대해서는 응답을 취소할 수 있다 | 응답 취소 |
| 20 | 시스템은 설문이 이미 종료된 경우에는 회원이 응답 내용을 수정하거나 취소할 수 없도록 제한해야 한다 | 응답 수정, 응답 취소 |
| 21 | 관리자는 설문 응답 통계 정보를 조회할 수 있다 | 설문 통계 조회 |
| 22 | 관리자는 검색 단위를 최근 1주일, 1개월, 1년 중에서 선택할 수 있다 | 검색 단위 조정 |
| 23 | 시스템은 선택된 기간 동안의 전체 설문 수, 종료된 설문 수, 총 응답 수를 출력해야 한다 | 설문 통계 조회 |

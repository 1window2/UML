Use Case Description

Use Case: 회원가입

Actor Action System Response
1. '회원가입' 버튼 클릭 2. 회원가입 화면으로 전환
3.1 필수 입력 정보(ID, 비밀번호, 이름, 전화번호, 메일 등)를 입력
3.2 '가입' 버튼 클릭 		4.1 입력된 데이터의 유효성을 검증
4.2 DB에 해당 데이터를 저장
4.3 회원가입 완료 메시지 출력
4.4 로그인 화면으로 전환
Alternative Courses
3a. 필수 입력 정보 누락: System은 누락된 항목을 강조하고, '필수 항목을 모두 입력해주세요' 메시지 출력
3b. ID 중복: System은 '이미 사용 중인 ID입니다' 메시지 출력

Use Case: 회원 탈퇴

Actor Action System Response
1. '회원 탈퇴' 메뉴 선택 2. 회원 탈퇴 페이지로 전환
3. '탈퇴' 버튼 클릭 4.1 회원의 모든 이용권한과 데이터 삭제
4.2 탈퇴 완료 메시지 출력
4.2 로그인 화면으로 이동
Alternative Courses
1a. 관리자 로그인: '회원 탈퇴' 버튼 비활성화
3a. 탈퇴 취소: System은 데이터 변경 없이 탈퇴 절차를 중단하고 이전 화면 유지
